# TasteTheory  – Fresh & Tasty Restaurant Landing Page

TasteTheory is a modern, single-page restaurant website built with **pure HTML, CSS and JavaScript**.  
It’s designed for a cafe/restaurant that wants a clean landing page with:

- Hero image slider
- Today’s specials
- Menu section
- Food gallery
- Table reservation form
- Simple Login & Signup mock pages (front-end only)

___________________________________________________________________________

##  Features

###  Hero Slider
- Auto-playing image slider on the home section
- Manual **Prev / Next** buttons
- Dots to jump between slides
- Pauses on hover
___________________________________________________________________________

###  Today’s Specials
- Highlight cards for featured dishes
- Each card shows image, description and price
- “Add” button (currently just shows an alert)
___________________________________________________________________________

###  Menu Section
- Grid layout of popular dishes  
- Each item shows:
  - Dish name  
  - Short description  
  - Price  
___________________________________________________________________________

###  Gallery
- Simple mini-gallery of food photos
- Great to show off real dishes from the kitchen
___________________________________________________________________________

### Reservation Form
- “Reserve a Table” form with:
  - Name
  - Email
  - Date & Time
  - Number of guests
  - Notes
- On submit: shows an alert and scrolls back to home  
  *(No backend yet – front-end only)*
___________________________________________________________________________

###  Login & Signup Pages
- Separate views for:
  - **Login** 
  - **Sign Up** 
- Basic validation (required fields, password length, terms checkbox)
- Currently show alerts (placeholders for real authentication later)

___________________________________________________________________________

##  Tech Stack

- **HTML5** – structure
- **CSS3** – custom responsive styling (no framework)
- **JavaScript** – slider, navigation, simple page switching
- **No backend / No build tools** – just open in a browser
